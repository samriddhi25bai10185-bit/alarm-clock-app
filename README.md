# ⏰ Smart Alarm Clock Pro

A modern desktop alarm clock application built with **Python**, **CustomTkinter**, **Pygame**, and **Plyer**. The application provides an intuitive dashboard-style interface for creating, managing, and monitoring alarms.

---

## ✨ Features

### ⏰ Alarm Management
- Create multiple alarms
- Custom alarm labels
- Delete individual alarms
- Clear all alarms
- Alarm status tracking

### 🔔 Alarm Functions
- Custom alarm tones (.mp3/.wav)
- Alarm notifications
- Stop alarm functionality
- Snooze support (5, 10, or 15 minutes)

### 🎨 Modern UI
- Dashboard-style interface
- Dark / Light mode toggle
- Responsive card layout
- Large live digital clock
- Date display
- Active alarm cards

### 💾 Data Persistence
- Automatically saves alarms to JSON
- Loads saved alarms on startup

### 🔊 Audio Controls
- Volume adjustment slider
- Continuous alarm playback until stopped

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/alarm-clock-app.git
```

### 2. Navigate to Project Folder

```bash
cd alarm-clock-app
```

### 3. Install Dependencies

```bash
pip install customtkinter pygame plyer
```

Or

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python main.py
```

## 🎯 How to Use

### Add an Alarm

1. Enter an alarm label.
2. Select Hour, Minute, and Second.
3. Choose an alarm tone.
4. Click **Add Alarm**.

### Snooze Alarm

When an alarm rings:

1. Click **Snooze**.
2. Select snooze duration.
3. Alarm will be rescheduled automatically.

### Stop Alarm

Click **Stop Alarm** to stop alarm playback immediately.
